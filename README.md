AI Football Simulator

Simple Python-based football (soccer) simulator where teams are controlled by AI.

Quickstart

1. Run the CLI example:

```bash
python cli.py
```

Files

- `src/team.py`: Team model and tactics
- `src/match.py`: Match simulation engine
- `src/league.py`: League manager and standings
- `cli.py`: Example runner that executes a round-robin league
